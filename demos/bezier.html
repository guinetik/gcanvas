The demo shows the separation between Shapes (visual definitions) and GameObjects (interactive entities).
BezierShape defines curve paths using commands like "M" (move), "L" (line), "Q" (quadratic), and "C" (cubic).
Control Points are wrapped with ShapeGOFactory to make them interactive, while guide lines show relationships between points.
Switch between Add Points mode to place new control points and Edit Points mode to drag and adjust the curve shape.

# GCanvas v1.0.3 Release Notes

## New Features

### SpriteSheet Class
Added a new `SpriteSheet` class for enhanced sprite management. Enables loading and rendering sprite sheets with frame-based animations.

### Boolean Algebra Utilities
New `BooleanAlgebra` module in the math library providing:
- Logic gate functions (AND, OR, NOT, XOR, NAND, NOR)
- Boolean expression parsing into AST
- Expression evaluation with variable assignments
- Truth table generation with Gray-code ordering

### Enhanced Input Handling
Added new key mappings to the `Keys` class:
- R, F, Z, C keys now available for gameplay controls

## New Demos
- Genuary 2026 project with interactive daily prompt visualizations
- New hyperbolic geometry demos
- Grid Bounce, Hex Grid, Circuit, Hex Bloom, Hex Flow demos
- Psychedelic Drop and Interference pattern demos
- 3D Logo demo
- StarFaux rail shooter game
